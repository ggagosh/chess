# Chess App

Base React application for the chess project, scaffolded from the current Vite
React TypeScript template and adjusted to match the requested modern stack:

- Vite 8
- `@vitejs/plugin-react`
- `oxlint`
- `oxfmt`

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run format
npm run format:check
npm run build
```

The app currently provides a branded starter shell for the future chess UI, plus
the linting, formatting, and build pipeline requested in `OME-118`.
